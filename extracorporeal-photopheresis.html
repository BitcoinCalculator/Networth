Therakos CELLEX — Treatment Tips
Reference: Therakos hotline (877) 566-9466. Always follow the laminated machine guide.
Standard Medications
NS 500 mL × 1
Heparin 10,000 units in 500 mL NS × 1 (20 units/mL)
Methoxsalen (Uvadex) 200 mcg in 10 mL — single-use vial
Pre-Procedure Considerations
CBC HCT checked within 48 h.
Patient must exercise UV precautions × 24 h: sunglasses, long sleeves, stay indoors.
ECV + threshold bag vs patient TBV — flag if > 15%.
CELLEX cassette volume: 240 mL.
Peripheral IV: hot packs / tourniquet to optimize flow.
Verify lamp life > 10 hours remaining.
Machine Setup (follow laminated guide)
Install tubing from the bottom up.
UV glass → hang bags → clip on cassette: C-R-Y (clear, red, yellow) → bowl.
Install tubing around motors — make sure tubing does not get pinched.
LAST: install the pressure pods (important).
Spike Saline (WHITE spike) and Heparin bag (ORANGE spike).
Connect Blue → Red as prompted.
Priming & Testing (auto ≈ 7 minutes)
1. "Prime Access" button.
2. Hit "Single" or "Double" needle to match access setup.
Machine Programming — verify "13 zeros"
Return Bag Threshold = 100 mL
Max speed collect = 50 mL/min
Max speed return = 50 mL/min
Collect Pressure Limit = 200
Return Pressure Limit = 200
Total blood processed = 1500 mL max
Starting Treatment & Phases
Start slow: Collect 25, Return 10.
COLLECT — Phase 1: Purging Air.
COLLECT — Phase 2: Drawing / Recirculating.
When you hear "BEEP," NET balance ≤ −100, OR blood processed > 300 mL → increase: Collect 30–40, Return 35–45.
Buffy coat phase: Open Graph → watch the S-wave (Hct rises) → look for "END BUFFY".
UVA – Pause – Elutriate – Empty Bowl – Rinse follow as prompted.
Methoxsalen (Uvadex) Mixing
Formula: Methoxsalen volume (mL) = treatment volume (mL) × 0.017
Inject Uvadex into the photoactivation bag per CELLEX prompt before UVA cycle.
Single-use 200 mcg / 10 mL vial. Discard remainder.
